Formulario de Registro

ID es requerido.
ID debe tener al menos 3 caracteres.
Nombre es requerido.
Nombre debe tener al menos 3 caracteres.
Descripción es requerida.
La descripción no debe superar los 200 caracteres.
Logo es requerido.
Fecha de Liberación es requerida.
Fecha de Restauración es requerida.
Fecha de Restauración debe ser posterior a la Fecha de Liberación.
